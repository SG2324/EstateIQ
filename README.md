**EstateIQ Mumbai** is a machine learning-powered real estate price estimator for the Mumbai property market, built using a Gradient Boosting Regressor trained on 3,000 properties across 30 localities, achieving 92% R² accuracy. It takes 8 inputs — carpet area, BHK, age, floor position, amenities, furnishing, Vastu facing, and locality — to generate instant price estimates in Lakh/Crore. The entire trained model (200 decision trees) is serialized to JSON and runs client-side via vanilla JavaScript, requiring no backend, no API, and no framework. The frontend is a single self-contained HTML file with a searchable Mumbai locality selector, real-time predictions, and area-specific price-per-sqft context spanning zones from Colaba to Panvel.
